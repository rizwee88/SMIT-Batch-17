### Operators & Expressions:

#### Arithmetic operators: +, -, \*, /, %, etc.:

JavaScript supports various arithmetic operators, including addition (+), subtraction (-), multiplication (\*), division (/), and modulus (%).

```js
// Addition
var sum = 2 + 3;

// Subtraction
var difference = 5 - 2;

// Multiplication
var product = 2 \* 3;

// Division
var quotient = 10 / 2;

// Modulus
var remainder = 10 % 3;
```
